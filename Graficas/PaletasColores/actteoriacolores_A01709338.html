Triada

In[4]:
# Crear una figura
plt.figure(figsize=(8, 6))

# Crear un grfico de barras horizontales con los colores
bar_width = 0.4  # Grosor de las barras
plt.barh(
    range(len(triada)),
    [1] * len(triada),
    color=triada,
    edgecolor="black",
    height=bar_width,
)

# Configurar los ejes
plt.yticks(range(len(triada)), triada)
plt.xticks([])
plt.title("Triada de Colores")

# Aadir los cdigos de color en el centro de cada barra
for index, value in enumerate(triada):
    plt.text(0.5, index, value, ha="center", va="center", color="black", fontsize=10)

# Mostrar la grfica
plt.show()

En un ambito de graficos, si se presta a visualizaciones gradientes para representar los mismos tipos de datos, pero para una visualizacin de diversos tipos de datos, no creo que sea la mejor opcin porque no hay tanto contraste entre los colores.
