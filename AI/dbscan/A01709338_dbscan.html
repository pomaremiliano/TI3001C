DBSCAN

El algoritmo DBSCAN (Density-based spatial clustering of applications with noise) es un algoritmo de agrupamiento de datos no supervisado que identifica grupos de puntos en un conjunto de datos basado en la densidad local de los puntos.

El algoritmo DBSCAN funciona de la siguiente manera:

  1. Para cada punto del conjunto de datos, se calcula el nmero de puntos a una distancia mxima de l. Esta zona se denomina -vecindad del punto.
  2. Un punto se considera un punto central si tiene al menos $k$ puntos en su -vecindad, incluida l mismo.
  3. Dos puntos centrales que se encuentren dentro de la -vecindad del otro se consideran miembros del mismo clster.
  4. Los puntos que no son puntos centrales se consideran anomalias.

El algoritmo DBSCAN tiene dos parmetros principales:

  • : la distancia mxima a la que se consideran vecinos dos puntos.
  • k: el nmero mnimo de puntos necesarios para que un punto se considere un punto central.

El algoritmo DBSCAN es un algoritmo verstil que puede ser utilizado para una amplia gama de aplicaciones, incluyendo:

  • Segmentacin de imgenes
  • Anlisis de datos espaciales
  • Deteccin de anomalas

Este algoritmo funciona bien si todos los clusters son lo suficientemente densos y estn bien separados por regiones de baja-densidad. La clase DBSCAN en Scikit-Learn es simple de usar. Probemos con el dataset moons:

        try: 
            sil_c = silhouette_score(X, dbscan.labels_)

                best_still, best_eps, best_min_pts = sil_c, e, mp
        except: 
            print('No se pudo calcular el Silhoutte score.')
